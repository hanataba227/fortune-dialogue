# fortune-dialogue
사담(四談, Fortune Dialogue) - AI 손님과 대화를 나누며 사주를 풀어가는 감성 대화형 웹 프로젝트

# 🪶 프로젝트 개요

이 프로젝트는 **사주를 보러 온 가상의 인물과 대화를 나누며, 그들의 이야기 속에서 사주를 해석하는 감성 대화형 상담 서비스**입니다.  
사용자는 웹 화면을 통해 등장인물과 일상적인 이야기를 주고받고, 그 대화의 흐름 속에서 자연스럽게 사주 상담이 진행됩니다.  
대화가 끝나면, 인물의 사연과 대화를 바탕으로 해석된 사주 결과가 전달되어 **대화를 통한 이해와 공감 중심의 사주 경험**을 제공합니다.

---

# 🌙 주요 기능

| 구분                 | 기능명                  | 상태 | 설명                                                                                                          |
| -------------------- | ----------------------- | ---- | ------------------------------------------------------------------------------------------------------------- |
| **인물 생성**        | 사주 방문자 프로필 생성 | ✅   | AI가 인물의 성격, 나이, 성별, 직업 등을 자동 생성하고 데이터베이스에 저장합니다.           |
| **대화 인터페이스**  | 실시간 대화형 상담      | ✅   | 사용자가 AI 인물과 자유롭게 대화하며, 인물의 성격과 고민이 반영된 자연스러운 대화가 이어집니다.           |
| **사주 해석 결과**   | 대화 기반 결과 생성     | ✅   | 대화 내용을 AI가 분석하여 운세, 성격, 조언을 포함한 사주 해석 결과를 생성합니다.                        |
| **기록 관리**        | 대화 및 결과 저장       | ✅   | 모든 상담 세션의 대화, 인물 프로필, 사주 결과가 Supabase에 저장됩니다. |
| **사용자 경험 강화** | 감성적 UI               | ✅   | 한국 전통 스타일의 아름다운 UI로 몰입감 있는 상담 경험을 제공합니다.                   |
