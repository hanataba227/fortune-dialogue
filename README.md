# fortune-dialogue
사담(四談, Fortune Dialogue) - AI 손님과 대화를 나누며 사주를 풀어가는 감성 대화형 웹 프로젝트

# 🪶 프로젝트 개요

이 프로젝트는 **사주를 보러 온 가상의 인물과 대화를 나누며, 그들의 이야기 속에서 사주를 해석하는 감성 대화형 상담 서비스**입니다.  
사용자는 웹 화면을 통해 등장인물과 일상적인 이야기를 주고받고, 그 대화의 흐름 속에서 자연스럽게 사주 상담이 진행됩니다.  
대화가 끝나면, 인물의 사연과 대화를 바탕으로 해석된 사주 결과가 전달되어 **대화를 통한 이해와 공감 중심의 사주 경험**을 제공합니다.

---

# 🌙 주요 기능 요약

| 구분                 | 기능명                  | 설명                                                                                                          |
| -------------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------- |
| **인물 생성**        | 사주 방문자 프로필 생성 | 인물의 성격, 나이, 성별, 직업 등 기본 속성을 자동 생성하고, 이를 기반으로 이미지가 함께 제작됩니다.           |
| **대화 인터페이스**  | 실시간 대화형 상담      | 사용자가 인물(LLM)과 자유롭게 대화하며, 일상적인 이야기를 나누는 동안 상담이 자연스럽게 이어집니다.           |
| **사주 해석 결과**   | 대화 기반 결과 생성     | 인물의 말투와 이야기 흐름을 분석해, 그 사람의 운세·성향·조언을 해석 형태로 제시합니다.                        |
| **기록 관리**        | 대화 및 결과 저장       | 각 상담 세션의 대화 내용, 인물 프로필, 생성된 이미지, 사주 결과가 함께 저장되어 추후 다시 확인할 수 있습니다. |
| **사용자 경험 강화** | 감성적 연출             | 등장 인물의 표정, 말투, 스토리 구성을 통해 실제 사람과 이야기하는 듯한 몰입감을 제공합니다.                   |
